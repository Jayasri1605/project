html:5

Hypertext markup language or HTML is a widely used computer language for developing websites and web applications. HTML developers create website layouts based on wireframes and templates and use Cascading Style Sheets (CSS) to create responsive web pages.

css

CSS literally means cascading style sheets and it represents a programming language designed to offer stylistic syntax to build websites and web applications. By contrast, CSS determines the stylistic properties of a website like colors, page layouts, and fonts. So HTML provides your functional framework, and CSS provides design and aesthetic elements. However, CSS interacts with HTML to perform its function of adding style elements.

JavaScript

In JavaScript programming language that allows Web Developers and Designers to create interactive web pages. JavaScript facilitates the creation of user-friendly websites and applications that allow for seamless interaction on web browsers. JavaScript enables users to access and interact with dynamic web content, including text, images, audio, and video for PDFs, widgets, and Flash applications. Company sites using JavaScript include Google, Yahoo, Amazon, eBay, and LinkedIn.
